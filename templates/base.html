Capstone Project @ The TKH 2020 Innovation Fellowship (Phase 1)

About This Project

I have scraped data from this website comparing stats on Maternity Leave by country for 2020. My scraping script targeted the data in the table on the website and cleaned it up to extract the specific information needed. I automated the process of storing the extracted data into my database. At last, I created a Flask Application that made my database accessible from the web, created app routes for 3 different views of my database, and deployed it on Heroku.

This is the link to the GitHub repo for this project.
